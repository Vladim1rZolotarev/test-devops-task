Для добавления изображения профиля:

1. Сохраните предоставленное изображение в этой директории с именем "profile.jpg"
2. Убедитесь, что файл имеет правильное расширение (.jpg)
3. Изображение будет автоматически отображаться на сайте

Примечание: Если вы хотите использовать другое изображение, просто замените файл profile.jpg на ваше изображение, сохранив то же имя файла.
