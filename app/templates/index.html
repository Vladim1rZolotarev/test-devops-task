О сайте

Этот сайт представляет собой полноценную DevOps-инфраструктуру для развертывания веб-приложения с использованием современных инструментов и практик. Проект демонстрирует мои навыки в области автоматизации, настройки CI/CD, мониторинга и обеспечения надёжности инфраструктуры.

Prometheus Alertmanager Grafana Loki Promtail Node Exporter

Архитектура проекта включает:

  • Веб-приложение на Flask с кэшированием через Redis
  • Балансировку нагрузки и проксирование через Nginx
  • Мониторинг всех компонентов через Prometheus и Grafana
  • Централизованное логирование через Loki и Promtail
  • Автоматическое развертывание через CI/CD (GitHub Actions)
  • Инфраструктуру как код с использованием Terraform
